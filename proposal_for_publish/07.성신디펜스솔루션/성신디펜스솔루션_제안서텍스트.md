## PAGE 3 — 시대+기술의 변화 & 홍보영상 개선방향

```
시대 + 기술의 변화
정보 / 5G · AI / 소통 / 자동화 / ESG
+ 차별성 / 통일성 / 편리성
-> 차별화되고 통합되는 시각적 콘텐츠 필요

[ 홍보영상 개선방향 ]

영상콘텐츠
- 방위기술 기반 에너지 사업 확장으로 진화하는 성신디펜스솔루션의 새로운 영상 콘텐츠 개발이 필요

기업정보
- 설비 : 부산·동해 공장 확장과 정밀 용접·특수 제조 장비 드론 샷을 통한 시각적 기술력과 규모감 제공
- 기술 : 방산혁신기업 100 선정·T3 기술역량·함정 63척 납품 실적을 통한 국방 기술 신뢰도와 차별성을 줌
- ESG : ISO 14001 환경경영 인증과 방위·에너지 산업 확장을 통한 지속가능 기술 기업 비전 제시

고객편의성
- 온라인·소셜미디어 채널을 통해 국방·에너지 분야 고객과 소통하고 글로벌 기술 파트너십을 확대하는 데 효과적
```

## PAGE 5 — 키워드 분석 + 신규 콘텐츠 도출

```
❷ 키워드 분석 + 신규 콘텐츠 도출

신규 콘텐츠 도출:
  · 해군·해경 함정 63척 탑재 IRSS 스텔스 기술
  · 군용·민수 헬기 155대 탑재 IRS & Ejector 시스템
  · 가스터빈 정밀 필터링·소음저감 GT Intake 솔루션
  · LS전선·LS일렉트릭 해저케이블·변압기 외함 신규 사업
  · GE·Parker Hannifin 등 미국 기술협약 글로벌 파트너십

Keyword (중앙 배치):  방위 기술과 에너지 산업의 융합

키워드 태그:
  변화 / 기술 혁신 / 신뢰 / 기술 선도
  방산기술 선도 / 정밀함 / 글로벌파트너십 / 첨단/자동화
```
