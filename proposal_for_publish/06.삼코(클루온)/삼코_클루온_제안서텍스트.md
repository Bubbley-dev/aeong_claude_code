## PAGE 3 — 시대+기술의 변화 & 홍보영상 개선방향

```
시대 + 기술의 변화
정보 / 5G · AI / 소통 / 자동화 / ESG
+ 차별성 / 통일성 / 편리성
-> 차별화되고 통합되는 시각적 콘텐츠 필요

[ 홍보영상 개선방향 ]

영상콘텐츠
- 정밀 제조 기술 기반 신규 B2C 브랜드 클루온 런칭으로 도약하는 삼코의 새로운 영상 콘텐츠 개발이 필요

기업정보
- 기술 : 특허 3건·100PPM 품질 인증·천만 불 수출탑을 통한 글로벌 정밀 제조 기술력과 차별성을 줌
- 제품 : 국내 생산·99.9% 살균·1년 무상 AS를 갖춘 클루온 UV 살균건조기의 소비자 위생 가치 시각화
- ESG : ESG 경영 선언과 국내 생산 클루온을 통한 지속가능 브랜드 비전 제시

고객편의성
- 온라인·소셜미디어 채널을 통해 B2B·B2C 고객 모두와 소통하고 새로운 생활위생 시장 개척에 효과적
```
