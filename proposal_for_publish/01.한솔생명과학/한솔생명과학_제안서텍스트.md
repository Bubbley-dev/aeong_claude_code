## PAGE 3 — 시대+기술의 변화 & 홍보영상 개선방향

```
시대 + 기술의 변화
정보 / 5G · AI / 소통 / 자동화 / ESG
+ 차별성 / 통일성 / 편리성
-> 차별화되고 통합되는 시각적 콘텐츠 필요

[ 홍보영상 개선방향 ]

영상콘텐츠
- 신사옥 2,000평 증축 및 우녹스 제조기계 독점권 확보로 도약한 생산 인프라를 알리는 새로운 영상 콘텐츠 개발이 필요

기업정보
- 제조 설비 : 신사옥·클린룸·최신 제조기계 드론 샷 및 특수장비를 통한 시각적 규모감과 신뢰도 제공
- 기술 : CGMP + ISO 7종 인증을 통한 글로벌 수준의 ODM/OEM 품질 경쟁력과 차별성
- ESG : 3천만불 수출 달성과 17개국 글로벌 진출을 통한 K-뷰티 파트너로서의 새로운 비전 제시

고객편의성
- 온라인·소셜미디어 채널을 통해 글로벌 고객과 소통하고 ODM/OEM 파트너십을 확장하는 데 효과적
```

**출처**
| 내용 | 출처 |
|------|------|
| 신사옥 2,000평 증축 | 역사 페이지 (2024년 항목) |
| 우녹스 그룹 제조기계 한국 독점권 확보 | 뉴스룸 (2026-05-11) |
| CGMP + ISO 7종 인증 | 인증현황 페이지 |
| ODM/OEM 솔루션 | 솔루션프로세스 페이지 |
| 3천만불 수출의 탑 + 대통령 표창 | 역사 페이지 (2025년), 뉴스룸 (2025-12-05) |
| 17개국 글로벌 박람회 참가 | 해외박람회 페이지 (참가국 집계) |

## PAGE 5 — 키워드 분석 + 신규 콘텐츠 도출

```
❷ 키워드 분석 + 신규 콘텐츠 도출

신규 콘텐츠 도출:
  · 스킨케어·헤어·바디·메이크업·의약외품 전 품목 ODM/OEM
  · CGMP·ISO 22716 등 7종 인증
  · 신사옥 2,000평 증축·우녹스 제조기계 한국 독점권 확보
  · 3천만불 수출의 탑·17개국 글로벌 박람회 참가
  · 연구기관 MOU·공동특허 등록

Keyword (중앙 배치):  K-뷰티와 제조 기술의 고도화

키워드 태그:
  변화 / 기술 혁신 / 신뢰 / 기술 선도
  ESG 지속가능 / 정밀함 / 글로벌파트너십 / 첨단/자동화
```
