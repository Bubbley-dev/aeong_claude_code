## 1. 입력 파일 규격

버블리님이 넘겨줄 파일:

| 파일 | 설명 |
|---|---|
| `input.hwp` | 내용 원본 (고객 니즈, 서비스 내용, 기대효과, 기간표, 역할표 등) |
| `img_1.jpg` ~ `img_9.jpg` | 포트폴리오 이미지 9장 (3×3 그리드) |

### 이미지 명명 규칙
```
img_1.jpg  img_2.jpg  img_3.jpg   ← 포트폴리오 그룹 1 (행1)
img_4.jpg  img_5.jpg  img_6.jpg   ← 포트폴리오 그룹 2 (행2)
img_7.jpg  img_8.jpg  img_9.jpg   ← 포트폴리오 그룹 3 (행3)
```

파일들을 `proposal/` 폴더에 넣고 아래 명령 실행:
```bash
cd /mnt/c/Users/ksh/Desktop/Claude
# 파이썬 스크립트로 PPT 생성 (추후 별도 스크립트 작성)
python3 proposal/fill_template.py
```

## 4. 수동 작업 필요 항목 ⚠️

다음 항목은 직접 PPT를 열어 수정해야 해요:

### 4-1. 사업수행 기간 표 행 수가 9행보다 다를 때
- 현재 템플릿: 9행 고정
- 행 수가 다르면: PowerPoint에서 표 선택 → 행 추가/삭제

### 4-2. 역할 표 연구원 수가 다를 때
- 현재 템플릿: 5행 (책임연구원 + 연구원 1~4,5) 고정
- 인원 수 변동 시: PowerPoint에서 표 직접 수정

### 4-3. 포트폴리오 행 그룹별 "링크 아이콘" 버튼
- PPT 내에 그룹 도형(링크 버튼 역할)이 있음
- 해당 도형에 하이퍼링크가 설정되어 있으면 수동으로 변경 필요
- 방법: 도형 우클릭 → 하이퍼링크 편집

### 4-4. 이미지 크기/위치 미세 조정
- 자동 교체 후 이미지 비율이 원본과 다를 경우 수동 조정
- 각 이미지 셀 크기: 약 2.34인치 × 1.32인치

## 5. fill_template.py 사용법

