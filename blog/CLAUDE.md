## 5. 말투 & 문체 규칙

### 기본 문체
- **경어체** 사용 (`합니다 / 습니다 / 니다`) — Key Copy는 명사구/어절 단위, 설명 문단은 경어체로 구분
- 친근하되 전문적인 톤
- 독자에게 직접 말 걸기: `"~하고 계신가요?"`, `"~하셨나요?"`
- **본문은 짧고 담백하게.** 이미지가 주인공이므로 설명 과다 금지. 분량 기준은 섹션 7 참고

### 설명 문단 문체 규칙
- **직설적으로 서술.** "무엇을 만들었는지, 어떤 효과가 있는지" 중심으로 구체적으로 작성
- Key Copy처럼 추상적·비유적 표현 금지. 설명은 정보 전달이 목적
- 리스트/불릿 포인트 사용 금지. 문장으로 자연스럽게 이어서 쓸 것

### 이미지 관련 금지 사항
- **이미지에 보이는 내용을 본문에 그대로 서술 금지.** 명함, 차량, 로고 등 이미지로 확인 가능한 내용은 본문에서 제거
- **포트폴리오에서 제작 과정 디테일 언급 금지.** 페이지 수, 가이드북 규모, 작업량 수치 등 내부 제작 정보는 작성하지 않음. 결과물의 활용·효과 중심으로 서술

### 이모지 활용
기본적으로 사용하지 않음. 꼭 필요한 경우에만 최소한으로 사용.

### 네이버 SEO 키워드 반복 기법
핵심 키워드를 **굵게** 처리 후 바로 다음 줄에 한 번 더 반복 (네이버 검색 최적화).

예시:
```
기업 홈페이지 제작 및 UIUX 설계의 중요성
기업 홈페이지 제작 및 UIUX 설계의 중요성도 함께 커지고 있습니다.
```
```
3D 기반 웹디자인과 인터랙션 UI/UX 설계
3D 기반 웹디자인과 인터랙션 UI/UX 설계를 중심으로 작업을 진행했습니다.
```

### Key Copy 스타일
> **Key Copy = 본문 소제목.** 같은 말이다. "소제목"이라고 부르더라도 Key Copy 규칙을 적용할 것.

**규칙: 명사구/어절 단위. 동사 서술어로 끝나는 문장형 절대 금지.**

❌ 틀린 예 (문장형)
```
고객의 이해를 돕는 3D 모델링을 제작했습니다.
기술을 쉽게 전달하는 것이 목표였습니다.
```

✅ 올바른 예 (카피라이팅)
```
고객의 이해를 돕는
3D 모델링

기업 이미지를 가장 먼저 보여주는 Key Visual
```

(Key Copy는 짧게, 이어지는 설명 문단 첫 문장에서 Key Copy 키워드 반복)

## 6. SEO 및 마무리 & CTA 패턴
### SEO 인사이트 규칙

CTA 직전 배치서비스 키워드 최소 2개 포함정보 전달 중심, 150~300자.

#### 서비스 키워드 예시

[홈페이지]기업 홈페이지 제작, 반응형 홈페이지, UIUX 디자인

[영상]기업 홍보영상 제작, 모션그래픽 영상, PR영상 제작

[카탈로그]전자카탈로그 제작, 제품 카탈로그

[SNS]SNS 마케팅, 카드뉴스 제작

```
그럼 오늘의 포스팅은 여기까지입니다.
[서비스명] 문의가 있으시다면,
언제든지 알리지오에게 연락주세요!
```

## 10. 제목 패턴

모든 포스트 공통 포맷:
`[SEO 키워드], [핵심 가치/특성 서술] | [브랜드명(영문명)] [작업 유형]`

- SEO 검색어를 맨 앞에 배치 (네이버 검색 최적화)
- 쉼표 뒤 핵심 가치/특성을 동사형으로 서술 (`담다`, `전달하다`, `풀어내다` 등)
- `|` 뒤에 브랜드명(영문명 병기 권장) + 작업 유형

### 예시
```
브랜딩 디자인, BI부터 50주년 기념영상까지 | 국립종자원 K-seed 브랜딩 제작
기업홍보영상 제작, 미래 모빌리티를 영상으로 말하다 | 남양넥스모 기업홍보영상
제품 카탈로그 제작, 강렬한 비주얼로 브랜드 파워를 담다 | FLEX 제품 카탈로그 디자인
카드뉴스 제작, 글로벌 바이오 기술을 SNS에 담다 | 진우바이오 카드뉴스 마케팅
쇼핑몰 홈페이지 제작, AI 비주얼로 완성한 브랜드 경험 | AI-LAB 오로베 쇼핑몰 홈페이지
AI 이미지 생성, 복잡한 솔루션을 한 컷에 담다 | AI-LAB DT 전자카탈로그 사례
```

## 12. 작업물 저장 경로 규칙

- 기본 저장 경로: `blog/번호.[카테고리]브랜드명/` 폴더 하위
- 초안(MD)과 최종본(HTML) 같은 폴더에 나란히 저장
- 예시:
  - `blog/08.[홈페이지]골프존클라우드/blog_post.md` (초안)
  - `blog/08.[홈페이지]골프존클라우드/blog_post.html` (최종본)
- 번호는 기존 폴더 순서에 맞춰 부여

### 작업 순서
1. 파일 목록 확인 후 계획 작성(계획 단계에서 이미지/파일을 읽거나 초안을 바로 작성하는 것 절대 금지)
2. MD 초안 작성 → 검토·수정
3. 확정 후 HTML 최종본 변환 저장

## 13. HTML 출력 규격

네이버 블로그 에디터 붙여넣기 최적화를 위해 `.html` 파일로 산출한다.
Chrome에서 열어 Ctrl+A → Ctrl+C → 네이버 에디터 Ctrl+V 방식으로 사용.

### 기본 구조 규칙

| 요소 | HTML 처리 |
|------|-----------|
| 전체 정렬 | `text-align: center` |
| 볼드 키워드 | `<strong>` |
| 섹션 제목 (Key Copy) | `<blockquote class="title">` — 상하 보더, italic 24px |
| 고객 인용문 | `<blockquote class="quote">` — 좌측 보더, italic 24px |
| 이미지 삽입 위치 | `<div class="img-placeholder">` — 파란 점선 박스 |
| 줄바꿈 | 모바일 기준 10~20자 내외로 `<br>` 수동 삽입 |
| 해시태그 | `.hashtags` 클래스, 14px |

### 이미지 프롬프트 포함 방법
이미지 프롬프트를 제공하는 경우, placeholder 안에 `<textarea>` 로 삽입.
클릭 한 번으로 전체 선택되고, Ctrl+A 전체 복사 시 textarea 내용은 자동 제외됨.

```html
<div class="img-placeholder">
  <span>🖼</span>
  <div class="label">이미지 N 삽입 — 설명</div>
  <div class="prompt-label">▼ ChatGPT 이미지 프롬프트 (클릭 후 Ctrl+A → Ctrl+C)</div>
  <textarea class="prompt-box" readonly rows="10" onclick="this.select()">
[프롬프트 내용]
  </textarea>
</div>
```
