## 3. 인삿말 패턴 (카테고리별)

카테고리마다 인삿말이 다름. 반드시 해당 카테고리 패턴을 따를 것.

### 홈페이지 / 케이스스터디 / 동영상
```
안녕하세요,
알리지오입니다!
```

### 정부지원사업
```
안녕하세요, 알리지오입니다.
```
(느낌표 없음, 공식적 톤)

### 알지비뉴스 (성과/후기/홍보)
```
안녕하세요!
성공적인 해외 진출을 위한 최고의 파트너, 알리지오입니다.
```
또는
```
안녕하세요!
글로벌 진출의 든든한 파트너, 알리지오입니다.
```

## 4. 전개 구조 (5단계 공식)

```
[1] 인사 + 주제 소개 (2~4줄)
     ↓
[2] 고객 페인포인트 또는 트렌드 언급 (문제 제기, 공감 유도)
     ↓
[3] 본론: 소제목 + 설명 + 시각 요소 설명 (이미지 캡션 형태)
     ↓
[4] 마무리 + CTA (문의 요청)
     ↓
[5] 해시태그 (15~20개)
```

## 5. 말투 & 문체 규칙

### 기본 문체
- **경어체** 사용 (`합니다 / 습니다 / 니다`)
- 친근하되 전문적인 톤
- 독자에게 직접 말 걸기: `"~하고 계신가요?"`, `"~하셨나요?"`

### 이모지 활용
- 알지비뉴스 카테고리: 이모지 적극 활용 (`⭐`, `🔥`, `✅`, `→`, `?`)
- 케이스스터디/홈페이지: 이모지 거의 없음, 깔끔한 텍스트 중심
- 정부지원사업: 이모지 최소화, 공식 문서 스타일 + 말미에 이모지 1~2개

### 네이버 SEO 키워드 반복 기법
핵심 키워드를 **굵게** 처리 후 바로 다음 줄에 한 번 더 반복 (네이버 검색 최적화).

예시:
```
기업 홈페이지 제작 및 UIUX 설계의 중요성
기업 홈페이지 제작 및 UIUX 설계의 중요성도 함께 커지고 있습니다.
```
```
3D 기반 웹디자인과 인터랙션 UI/UX 설계
3D 기반 웹디자인과 인터랙션 UI/UX 설계를 중심으로 작업을 진행했습니다.
```

### 불릿 포인트 활용
3개 전후의 불릿으로 핵심 정리:
```
- 복잡한 구조를 시각적으로 표현
- 보이지 않는 개념을 직관적으로 전달
- 핵심 메시지를 짧은 시간 안에 전달
```

### 소제목 스타일
```
고객의 이해를 돕는
3D 모델링
```
또는
```
기업 이미지를 가장 먼저 보여주는 Key Visual
```
(소제목은 짧게, 이어지는 설명은 첫 문장에서 소제목 키워드 반복)

## 13. 자동화 포스팅 작성 가이드 (프롬프트 입력 규격)

자동화 시 아래 항목을 입력으로 받아 포스팅 생성:

```yaml
카테고리: [홈페이지|케이스스터디|동영상|정부지원사업|알지비뉴스]
제목: ""
포스팅_유형: [사례소개|서비스소개|정보전달|시리즈N편]
고객사명: ""   # 사례 포스팅 시
프로젝트_핵심: ""  # 1~2줄 요약
주요_서비스_키워드: []  # 해시태그에 활용
시리즈_정보:   # 시리즈 포스팅 시
  현재편: 1
  전체편수: 3
  다음편_주제: ""
```

### 자동화 작성 순서
1. 인삿말 (카테고리별 패턴 적용)
2. 포스팅 소개 (프로젝트/주제 1~2문장)
3. 페인포인트 또는 트렌드 (공감대 형성)
4. 본론 소제목 2~4개 + 각 설명
   - 핵심 키워드 굵게 + 반복 적용
   - 불릿 포인트 적절히 삽입
5. 마무리 + CTA (카테고리별 패턴 적용)
6. 해시태그 (3그룹 구성 규칙 적용)
