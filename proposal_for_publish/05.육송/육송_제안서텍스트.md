## PAGE 3 — 시대+기술의 변화 & 홍보영상 개선방향

```
시대 + 기술의 변화
정보 / 5G · AI / 소통 / 자동화 / ESG
+ 차별성 / 통일성 / 편리성
-> 차별화되고 통합되는 시각적 콘텐츠 필요

[ 홍보영상 개선방향 ]

영상콘텐츠
- 전기차 화재 확산방지 시스템 혁신제품 지정으로 도약한 소방 기술력을 알리는 새로운 영상 콘텐츠 개발이 필요

기업정보
- 소방 설비·시공 : 전기차 화재 직수장치·소방시공 현장 드론 샷을 통한 시각적 기술력과 현장감 제공
- 기술 : 소방청 혁신제품 지정·NEP 인증을 통한 공공 신뢰 기술력과 차별성을 줌
- ESG : 베트남 K-소방용품 총판 계약을 통한 글로벌 소방 파트너로서의 새로운 비전 제시

고객편의성
- 온라인·소셜미디어 채널을 통해 소방 안전 고객과 소통하고 새로운 시장 개척에 효과적
```

## PAGE 5 — 키워드 분석 + 신규 콘텐츠 도출

```
❷ 키워드 분석 + 신규 콘텐츠 도출

신규 콘텐츠 도출:
  · 전기차 화재 상방향 자동 직수장치·소방청 혁신제품 지정
  · ISO 9001·14001·45001·NEP 인증
  · 국무총리표창 3회·서울국제발명전시회 금상
  · 베트남 퍼비스글로벌 K-소방용품 총판 계약
  · 소방시공·열선시공 종합 솔루션

Keyword (중앙 배치):  소방 제조와 시공 기술의 융합

키워드 태그:
  변화 / 기술 혁신 / 신뢰 / 기술 선도
  공공안전 지속가능 / 정밀함 / 글로벌파트너십 / 첨단/자동화
```
