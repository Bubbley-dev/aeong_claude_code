## PAGE 3 — 시대+기술의 변화 & 홍보영상 개선방향

```
시대 + 기술의 변화
정보 / 5G · AI / 소통 / 자동화 / ESG
+ 차별성 / 통일성 / 편리성
-> 차별화되고 통합되는 시각적 콘텐츠 필요

[ 홍보영상 개선방향 ]

영상콘텐츠
- 신규 브랜드 모지에프 런칭 및 줄기세포 코스메슈티컬 라인 확장을 알리는 새로운 영상 콘텐츠 개발이 필요

기업정보
- 연구 현장 : 줄기세포 배양액 연구·임상 현장 촬영을 통한 과학적 신뢰감과 브랜드 전문성 시각화
- 기술 : 제대혈 줄기세포배양액 기반 독자 기술로 코스메슈티컬 시장에서의 차별성을 줌
- ESG : 대한적십자사 ESG실천기업 선정·산학협력 MOU로 지속가능한 바이오 브랜드 비전 제시

고객편의성
- 온라인·소셜미디어를 통해 병의원·소비자 양쪽에 줄기세포 과학 근거를 전달하고 신규 시장 확장에 효과적
```

**출처**
| 내용 | 출처 |
|------|------|
| 신규 브랜드 모지에프 런칭 | 뉴스활동 (2025 서울뷰티위크, 모지에프 헤어토닉 촬영) |
| E-SC60 리뉴얼 출시 | 뉴스활동 (제대혈줄기세포배양액 83만ppm E-SC60 리뉴얼) |
| 베트남 수출 계약 체결 | 회사연혁 (2025년) |
| ISO22716 인증 | 회사연혁 (2024년) |
| 제대혈 줄기세포배양액 | 제품소개 E-SC60 페이지 |
| 대한적십자사 ESG실천기업 | 회사연혁 (2025년) |
| 산학협력 MOU | 뉴스활동 (국립강릉원주대 MOU) |

## PAGE 5 — 키워드 분석 + 신규 콘텐츠 도출

```
❷ 키워드 분석 + 신규 콘텐츠 도출

신규 콘텐츠 도출:
  · 병의원·홈케어 코스메슈티컬 풀라인
  · 제대혈 줄기세포배양액 83만ppm E-SC60 리뉴얼 출시
  · ISO22716 인증·산학협력 MOU
  · 신규 브랜드 모지에프 런칭·베트남 수출 계약
  · 식약처 허가 의료기기·ODM/OEM 사업

Keyword (중앙 배치):  바이오 기술과 뷰티의 융합

키워드 태그:
  변화 / 기술 혁신 / 신뢰 / 기술 선도
  ESG 지속가능 / 임상검증 / 글로벌파트너십 / 첨단/바이오
```

**출처**
| 내용 | 출처 |
|------|------|
| 병의원 전용 스킨부스터~홈케어 풀라인 | 사업소개 페이지, 메인 페이지 |
| 제대혈 줄기세포배양액 83만ppm E-SC60 | 뉴스활동 (E-SC60 리뉴얼 보도자료) |
| ISO22716·산학협력 MOU | 회사연혁 (2024년), 뉴스활동 (강릉원주대 MOU) |
| 모지에프 런칭·베트남 수출 | 뉴스활동 (서울뷰티위크), 회사연혁 (2025년) |
| 식약처 허가 의료기기·ODM/OEM | 사업소개 페이지 (Medical Device, ODM/OEM 사업) |
| Keyword "줄기세포 재생과학의 새로운 기준" | 경영이념(비전) + 제품소개 종합 |
