## PAGE 3 — 시대+기술의 변화 & 홍보영상 개선방향

```
시대 + 기술의 변화
정보 / 5G · AI / 소통 / 자동화 / ESG
+ 차별성 / 통일성 / 편리성
-> 차별화되고 통합되는 시각적 콘텐츠 필요

[ 홍보영상 개선방향 ]

영상콘텐츠
- 글로벌AI로봇제조센터 구축과 AI 로봇 라인업 확장을 알리는 새로운 영상 콘텐츠 개발이 필요

기업정보
- 설비 : i-Series·SMAR 작동 씬과 드론 샷 등 특수 장비를 통한 시각적 기술력과 규모감 제공
- 기술 : 특허 18건·인증 19건·CES 3년 연속을 통한 AI 로봇 기술 신뢰도와 차별성을 줌
- ESG : 글로벌 투자유치와 8개국 진출을 통한 파워오토로보틱스의 새로운 비전 제시

고객편의성
- 온라인·소셜미디어 채널을 통해 스마트공장 자동화 도입 제조사와 소통하고 글로벌 파트너십을 확대하는 데 효과적
```

| 내용 | 출처 |
|------|------|
| AI 무인화 공정 로봇 / 스마트공장 자동화 선도 | 슬로건 "FIRST MOVER of Smart Factory Automation Machine", 소프트웨어 페이지 |
| i-Series·SMAR 실제 작동 씬 | i-Series 페이지, AMR 페이지 |
| 아산 글로벌AI로봇제조센터 | 연혁 (2025.09) |

## PAGE 5 — 키워드 분석 + 신규 콘텐츠 도출

```
❷ 키워드 분석 + 신규 콘텐츠 도출

신규 콘텐츠 도출:
  · 국내 유일 하이브리드 이형 부품 삽입 로봇 i-Series
  · PAIS 딥러닝 이형 전자부품 300개+ 무인 정밀 삽입
  · SMAR 모듈형 AMR 스마트공장 완전 무인화
  · CES 3년 연속·美연방의회 표창 수상 이력
  · 8개국 글로벌 진출·아산 글로벌AI로봇제조센터

Keyword (중앙 배치):  AI 로봇 기술과 스마트 제조의 융합

키워드 태그:
  변화 / 기술 혁신 / 신뢰 / 기술 선도
  AI 무인화 / 정밀함 / 글로벌파트너십 / 첨단/자동화
```
